# [字典](#dict)
1.字典的概念  
&emsp;字典是一种映射关系的类型，字典具有可变性，可以像列表那样增加或者减少.  
2.映射操作  
&emsp;
``` python
    >>> D={'food':'Spam','quantity':4,'color':'pink'} #创建一个字典对象
    >>> D['food']       # 匹配键为food的字典值
    'Spam'
    >>> D['quantity']+=1
    >>> D
    {'food':'Spam','quantity':5,'color':'pink'}
````
